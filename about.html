About

This is a proxy generator for the card game Magic: The Gathering. Built with jQuery, it parses user input and retrieves card information from one open API before querying another for the images. It then displays the images in a grid. The images are appropriately sized for print, and specific CSS for printing is used to hide other page elements.

The program notes invalid input and displays it in a list above the results returned for the portion of input that was valid, allowing the user to edit just that input and run the program again.
